An AWK-ward primer

8/24/2023


If you're a fan of old languages, or if you just want to write easy and fasts script that can parse text line by line (for example, CSV files), then learning AWK might be for you! I'll skip the history to let us get straight into the meat of learning the tool/language, although you should check out wikipedia to learn more. By default, some flavor of AWK is included on macOS and most Linux distributions. Depending on which version you have, some features may be included or missing, but I'll be covering the base language in this post.

To start with, let's assume we have a csv file that we want to process in some way. It might contain a column of names as strings and a column of ages as integers, like such:

Andrew,22
Bob,34
Caroline,90
Daniel,59
...
